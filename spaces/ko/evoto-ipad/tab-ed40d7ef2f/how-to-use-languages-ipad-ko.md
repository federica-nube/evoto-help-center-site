#### 다국어 지원

계정 설정에 추가된 언어는 베트남어, 일본어, 한국어 및 번체 중국어입니다.

#### 기본 언어 설정

Evoto 아이패드 앱을 처음 다운로드하면, 시스템의 기본 언어 설정을 자동으로 따릅니다. 만약 시스템 언어가 Evoto에서 지원되지 않는 경우(예: 네덜란드어), 앱은 자동으로 영어를 기본 언어로 사용합니다. Evoto 아이패드 내에서 특정 언어를 직접 설정한 경우에는, 시스템 언어를 변경하더라도 앱에 설정된 언어는 바뀌지 않습니다. **언어 변경 방법** Evoto iPad에서 언어를 변경하는 방법은 다음과 같습니다. 설정 → 계정 설정 → 언어 변경으로 이동합니다. 시스템의 언어 설정 화면으로 이동하며, 원하는 앱 언어를 선택할 수 있습니다.

![](https://support.evoto.ai/wp-content/uploads/2025/04/91A8DBA0-9003-4CA8-935E-915410A206CF-1.png)

![](https://support.evoto.ai/wp-content/uploads/2025/04/IMG_0084.jpg)

**Evoto 내에서 지원 언어를 모두 보려면, 기기의 시스템 언어 설정에 2개 이상의 언어가 추가되어 있어야 합니다. 시스템 언어가 하나만 설정되어 있는 경우, 추가로 언어를 설정해야 모든 언어 옵션이 나타납니다.**
