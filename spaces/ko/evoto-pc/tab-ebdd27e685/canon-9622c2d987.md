**컴퓨터 설정 확인** Mac 15.0 이상의 컴퓨터 시스템에서는 개인정보 보호 설정에서 파일 및 폴더 권한을 활성화해야 합니다 -> 삭제 가능한 파일 권한. ![](https://support.evoto.ai/wp-content/uploads/2025/05/image-300x261.jpg)![](https://support.evoto.ai/wp-content/uploads/2025/05/image1-300x261.jpg)

**캐논 카메라 설정 확인** **절전 모드**를 끄세요 (절전 모드가 켜져 있으면 카메라는 일정 시간 동안 작동하지 않을 경우 절전 모드로 들어가며 연결이 끊어질 수 있습니다). Canon EOS R6을 예로 들어 설명하자면, **절전 설정**을 켜고 **디스플레이 꺼짐 시간**을 **최대치**로 설정한 후 **자동 전원 끄기**를 '**비활성화**'로 선택하세요. ![](https://evoto.gitbook.io/~gitbook/image?url=https%3A%2F%2F3099291537-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FGcGHhe6Z57mAjGmYtrQ4%252Fuploads%252FSx4N7uyP8ZfsRTKqsXxA%252Fimage.png%3Falt%3Dmedia%26token%3D49bf0265-da91-4105-9b96-74da054ad646&width=768&dpr=4&quality=100&sign=6f98a52a&sv=2) **'에코 모드'**를 '**끔**'으로 설정하세요. ![](https://evoto.gitbook.io/~gitbook/image?url=https%3A%2F%2F3099291537-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FGcGHhe6Z57mAjGmYtrQ4%252Fuploads%252FZ9ZzcRyRI0LFTMvDoCiH%252Fimage.png%3Falt%3Dmedia%26token%3D919f42c2-4ed5-4828-8218-a6a9830d0f09&width=768&dpr=4&quality=100&sign=bb5f4c32&sv=2) R6 II, R8, R5 M2와 같은 일부 카메라는**USB 연결 설정**에서 '**사진 가져오기/원격 제어**'로 설정이 필요합니다. ![](https://evoto.gitbook.io/~gitbook/image?url=https%3A%2F%2F3099291537-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FGcGHhe6Z57mAjGmYtrQ4%252Fuploads%252Fz9u0Qu4yCf9RxiSrVBjX%252Fimage.png%3Falt%3Dmedia%26token%3Ddaf5c0fb-9e5c-4f40-8aae-6de7c0414e07&width=768&dpr=4&quality=100&sign=c5dac9fb&sv=2) Canon 5D4와 같은 일부 카메라는 정상적인 연결을 위해 컴퓨터의 USB 3.0 이상 표준 인터페이스에 연결해야 합니다. USB 2.0 인터페이스에 연결할 경우, 연결 속도가 느리거나 실시간 미리보기를 활성화할 수 없으며 이미지 전송이 느려지는 등의 문제가 발생할 수 있습니다. USB 3.0 아이콘 ![](https://evoto.gitbook.io/~gitbook/image?url=https%3A%2F%2F3099291537-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FGcGHhe6Z57mAjGmYtrQ4%252Fuploads%252FSr2iKqOv12eK7Lk7yspq%252Fimage.png%3Falt%3Dmedia%26token%3De5a79cc1-4517-4349-902b-2cf1427e0a4b&width=77&dpr=4&quality=100&sign=a944dd6f&sv=2) 을 찾거나 USB 인터페이스의 색상을 확인해 보세요. 파란색![](https://evoto.gitbook.io/~gitbook/image?url=https%3A%2F%2F3099291537-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FGcGHhe6Z57mAjGmYtrQ4%252Fuploads%252F2SXhDs1CmznC2B1mMU6L%252Fimage.png%3Falt%3Dmedia%26token%3D7d812c18-1a22-4ef2-b2c3-e79d67c66fd1&width=83&dpr=4&quality=100&sign=3e712991&sv=2)이면 USB 3.0입니다 컴퓨터의 USB-Type C 인터페이스를 사용하여 테더 촬영을 할 수 있습니다.
