컴퓨터 설정 확인 Mac 15.0 이상 시스템에서는 '개인정보 보호' 설정에서 '파일 및 폴더 접근 권한'을 켜야 합니다. 여기서 '삭제 가능한 파일 권한'을 설정하세요.

![](https://support.evoto.ai/wp-content/uploads/2025/05/image-300x261.jpg)

![](https://support.evoto.ai/wp-content/uploads/2025/05/image1-300x261.jpg)

니콘 카메라 설정 확인 카메라에서 USB 데이터 연결 모드를 MTP/PTP로 설정하세요. **네트워크 메뉴 → USB 데이터 연결 → MTP/PTP**. USB 연결 우선순위를 촬영으로 설정하세요. **설정 메뉴 → USB 연결 우선순위 → 촬영.** USB 전원 공급을 끄세요. **메뉴 -> 설정 메뉴 (작은 렌치 아이콘) -> USB 전원 공급 -> 끔.** 니콘 Z7II, Z6III 및 다른 모델의 경우, USB 충전 포트와 데이터 전송 포트가 동일합니다. USB 전원 전송을 켜면 데이터 전송이 불안정해질 수 있으므로 꺼야 합니다.
