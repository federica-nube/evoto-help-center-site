# **컴퓨터 설정 확인**

For Windows System For MacOS Operation System Windows 시스템에서는 카메라가 올바른 연결 모드로 설정되어 있는지 확인해 주세요.![](https://res.evoto.ai/wordpress/8d54120d-94d3-4b78-9f6d-30239bfd8a41.png) MacOS 시스템에서는 올림푸스 카메라가 맥 컴퓨터와 안정적으로 연결되지 않을 수 있습니다. 더 안정적인 테더링을 연결하기 위해 카메라를 확장 USB C 허브에 연결한 후 컴퓨터에 연결하는 것을 권장드립니다. 올림푸스의 드라이버 소프트웨어를 웹사이트에서 다운로드하여 연결해 주세요.: [https://zadig.akeo.ie/](https://zadig.akeo.ie/)![](https://res.evoto.ai/wordpress/70d32bb3-9e62-467f-a4fc-51d69b325cf1.png) ![](https://res.evoto.ai/wordpress/b2200175-9a35-4db7-b2a4-6bd072d0a35d.png) 다운로드 후에는 옵션 -> 모든 장치 목록 -> 해당하는 카메라 모델을 찾아주세요.![](https://res.evoto.ai/wordpress/05edd4fa-dbce-488d-ba2b-8e1b859be410.png) ![](https://res.evoto.ai/wordpress/9fbb8c78-ca54-40ae-909d-d11c88a3e1a7.png) 드라이버를 교체할 때까지 클릭하여 설치를 완료하세요.![](https://res.evoto.ai/wordpress/a367aca0-f2ae-4060-959e-e0514c172756.png) ![](https://res.evoto.ai/wordpress/bde9fed6-35ce-4f52-9694-391c640e4529.png) **장치 관리자**에서**휴대용 장치** 아래에서 카메라가 컴퓨터에 연결되었는지 확인하세요.![](https://res.evoto.ai/wordpress/394c21ce-8384-4ba7-8f10-998cb226a285.png) **올림푸스 카메라 설정 확인** 단계1. **설정**으로 이동 -> **디스플레이 (D4)**-> **USB 모드 선택** -> "**USB / 테더 연결**" 선택 ![](https://res.evoto.ai/wordpress/b5c38deb-bc32-4eb9-93d8-10605cb7aba3.png) ![](https://res.evoto.ai/wordpress/94e96964-24cd-4e20-a8a4-88f1051048bb.png) 단계2. **설정**으로 이동 -> **유틸리티**-> **백라이트 LCD**로 이동하여 "**Hold**"로 설정해 주세요. ![](https://res.evoto.ai/wordpress/60ca846e-9ef1-428f-bd66-b79621fe7134.png) 단계3. **설정**으로 이동 -> **유틸리티** ->**슬립**으로 이동하여 "**Off**"로 설정해 주세요. ![](https://res.evoto.ai/wordpress/5c5be5b1-027f-4c32-bd5a-8179b63e4ea2.png) 단계4. **설정**으로 이동 -> **유틸리티** -> **자동 전원 꺼짐**으로 이동하여 "**Off**"로 설정해 주세요. ![](https://res.evoto.ai/wordpress/70f3dbf2-6e7e-46d0-b468-20e04f0aaea1.png) 단계5. **설정**으로 이동 -> **유틸리티** -> **퀵 슬립 모드**로 이동하여 "**Off**"로 설정해 주세요. ![](https://res.evoto.ai/wordpress/6d438f0f-7f3a-43d8-b69a-961ca2186b9e.png)
