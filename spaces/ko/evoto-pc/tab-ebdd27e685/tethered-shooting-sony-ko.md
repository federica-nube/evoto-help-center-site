# 테더링 촬영 설정 - 소니 카메라

컴퓨터 설정 확인 For Windows System For MacOS Operation System Windows 시스템의 경우, 장치 관리자로 이동하여 휴대용 장치 아래에서 카메라가 컴퓨터에 연결되었는지 확인합니다. MacOS 시스템의 경우, "사진" 앱을 열고 "장치" 아래에서 카메라가 컴퓨터에 연결되었는지 확인하십시오. ![](https://res.evoto.ai/wordpress/d0ad949d-fb16-404d-ab58-78a93a380483.png) ![](https://res.evoto.ai/wordpress/fe6c0d3a-e340-4b00-a855-47ffc80dbb8d.png) 소니 카메라 설정 확인 **USB 연결 모드** 소니 카메라는 USB 연결 모드 → PC 원격 아래에서 원격 연결 유형을 설정해야 합니다. **유형 1 - 소니 카메라의 새로운 버전** 컴퓨터에 연결한 후 카메라가 자동으로 연결 옵션을 팝업합니다. 원격 제어를 선택하십시오. (카메라에서 선택을 위한 이미지 팝업 필요) **유형 2 - 소니 카메라의 이전 버전** 한동안 출시된 카메라로, 최신 시스템을 지원하지 않습니다. 설정 → USB → USB 연결 모드 → PC 원격 선택합니다. ![](https://res.evoto.ai/wordpress/ce465daf-11e8-43c5-ab1f-8c581c033afb.png) ![](https://res.evoto.ai/wordpress/9c5d6783-f733-424b-9a83-060d25c2263f.png) **지정된 파일 저장 위치** 컴퓨터 또는 "지정된 파일 위치 + 카메라 (컴퓨터와 카메라에 모두 저장)" 컴퓨터에 연결한 후 사진 파일 저장 위치를 설정해야 합니다. 메뉴로 이동하여 네트워크 → 연결 / PC 원격 → 원격 촬영 설정으로 이동합니다: 정지 이미지 저장 위치: 목적지 + 카메라 (권장) 이미지 크기 저장: 원본 (필수) RAW+J 저장 이미지: RAW 및 JPEG (권장) ![](https://res.evoto.ai/wordpress/1b9b6515-d537-4cb0-b75f-beeed4d48396.png) ![](https://res.evoto.ai/wordpress/693a85d1-0150-4372-9c76-fb79e63915e4.png) ![](https://res.evoto.ai/wordpress/8e1a38f1-ce84-4ce8-9ddd-bddac88dbbaf.png) **Remote Shoot Setting** **버전 - 하드웨어 업그레이드** 예를 들어 Sony ILCE-7RM5의 경우, 먼저 카메라 소프트웨어 버전을 확인하고 최신 버전으로 업데이트합니다.
