### 테더링 촬영 설정 - 후지 카메라

컴퓨터 설정 확인 For Windows System For MacOS Operation System Windows 시스템의 경우, 장치 관리자로 이동하여 휴대용 장치 아래에서 카메라가 컴퓨터에 연결되었는지 확인하십시오. MacOS 시스템의 경우, 사진 앱으로 이동하여 장치 아래에서 카메라가 컴퓨터에 연결되었는지 확인하십시오.

![](https://res.evoto.ai/wordpress/ba9c16be-555a-42ee-bff9-14d34eb885b0.png)

![](https://res.evoto.ai/wordpress/42539357-7612-46af-b962-fb9864fc234d.png)

후지 필름 카메라 설정 확인 단계 1. 설정으로 이동하여 연결 설정을 찾습니다.

![](https://res.evoto.ai/wordpress/f80d4e21-f565-4a2c-bd8c-b40f41a20931.JPG)

단계 2. 연결 설정에서 연결 모드를 확인하고 USB 테더 촬영 AUTO로 변경합니다.

![](https://res.evoto.ai/wordpress/114f1442-fd48-44d1-a03c-8fc863f77fe2.JPG)

Check your Connection Mode

![](https://res.evoto.ai/wordpress/87171560-18d3-4cac-b101-5078ba3c5519.JPG)

USB tether shooting AUTO 단계 3. 연결 설정에서 USB 전원 공급 설정을 확인하고 설정을 OFF로 변경합니다.

![](https://res.evoto.ai/wordpress/82453ffd-b9cc-4ac4-95de-d008c986328b.JPG)

USB Power Supply Setting

![](https://res.evoto.ai/wordpress/392ded6d-4195-431c-b354-bf44ea64e8da.JPG)

**Turn OFF** the USB Power Supply Setting
