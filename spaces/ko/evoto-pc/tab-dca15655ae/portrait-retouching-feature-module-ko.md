**Evoto 인물 사진 보정 교육 영상**

Evoto는 이미지 파일이 가져와진 후에 자동으로 분석하며, 인물 사진 보정 기능 모듈 내에서는 이미지에서 사람들의 성별과 나이를 구별하여 자신만의 캐릭터 태그를 형성합니다. Evoto의 인물 사진 보정 기능은 서로 다른 캐릭터 성별과 나이에 대한 수정이 가능하며, 완료될 때 성별 또는 나이 수정을 기반으로 한 캐릭터의 일괄 동기화를 지원합니다. 일괄 동기화 기능을 통해 사용자는 한 번에 여러 이미지를 처리할 수 있습니다. 이는 작업 효율성을 향상시킬 뿐만 아니라 수정과 조정을 아티팩트 정확도로 빠르게 적용합니다. 또한 단일 인물의 조정과 단일 인물 수정 결과의 일괄 동기화도 지원합니다. Evoto의 지능적인 최적화 접근 방식은 각 사진이 최상의 시각적 효과를 위해 최적화되는 동시에 사진 속 인물의 자연스러움과 현실감을 유지합니다. **얼굴 인식** Evoto는 일반적으로 사진을 기반으로 얼굴 인식을 하고 캐릭터의 속성을 정의합니다. 그러나 때로는 사용자가 캐릭터의 속성을 수동으로 변경하길 원할 수 있습니다. 이러한 경우 캐릭터의 속성을 수동으로 변경하는 방법은 다음과 같습니다. ![](https://res.evoto.ai/wordpress/cb5170b3-6319-4567-a2ce-66af50e639b5.png) **인물 사진 보정 기능** 모듈로 들어간 다음 제어판의 오른쪽 상단 모서리에 있는 **얼굴 인식**을 클릭합니다. **캐릭터 추가와 삭제** ![](https://res.evoto.ai/wordpress/b10c7b83-c7e4-4c55-ad42-6c080739a844.png) 이미지에 새 캐릭터를 추가할 수 있습니다. "얼굴 위치 찾기" 기능을 선택하면 새로운 캐릭터를 이동하고 위치를 지정할 수 있는 프레임이 형성됩니다. 이 프레임을 사용하여 위치를 지정한 캐릭터를 Evoto AI가 자동으로 식별합니다. ![](https://res.evoto.ai/wordpress/ebc537b1-da13-4bec-b249-32b650a94956.png) 캐릭터를 삭제하려면 해당 캐릭터를 클릭해야 합니다(선택되면 노란색으로 강조됩니다). 그런 다음 삭제 옵션을 사용하여 해당 사진에서 캐릭터를 삭제할 수 있습니다. **인물 특성 변경** ![](https://res.evoto.ai/wordpress/20ea6bdb-1e6e-4a3b-a6c8-a56673fe61a9.png) 수정해야 할 캐릭터의 속성을 클릭합니다. 수정 후에는 새로운 수정 결과를 일괄 동기화할지 여부를 선택할 수 있습니다. "동기화"를 선택하면 프로젝트의 나머지 사진을 수정할 필요가 없으며, Evoto가 자동으로 선택한 최신 속성을 프로젝트의 나머지 사진에 동기화합니다. ![](https://res.evoto.ai/wordpress/257318db-dba9-4cc8-85ef-748c402f876b.png) 어린이의 경우, Evoto는 "어린이"과 "영유아" 선택을 지원합니다. 이미지에서 캐릭터가 "영유아"로 식별되면 "결점 제거" 기능 그룹 내의 "영유아 몸체 결함" 기능을 사용할 수 있게 됩니다. **개별 인물 보정** ![](https://res.evoto.ai/wordpress/182ebf9f-d90e-463c-af68-f56bce3ff4c5.png) 단체 사진에 대한 전체 인물 향상을 완료한 후에는 개별 편집 도구를 사용하여 단일 인물의 특성 효과를 미세 조정할 수 있습니다. 조정하려는 캐릭터를 선택하고, 해당 개인 캐릭터의 특정 부분을 변경하려면 원하는 특성 슬라이더를 드래그합니다. 일괄 동기화도 동일 프로젝트 내에서 동일한 캐릭터에 대한 개별 편집을 동기화하는 것을 지원합니다. 개별 편집은 모든 슬라이더의 '성별/연령 조정' 값을 다음에 사용할 수 있도록 저장합니다. **개별 편집 중 대상 인물의 인물 사진 보정이 이미 완료되었으므로 성별/연령 조절을 다시 진행하면 새 편집 내용이 적용되지 않습니다.개별 편집을 초기화하면 언제든지 관련 효과가 모두 지워지고 현재 성별/연령 조정 효과만 유지됩니다.** ![](https://res.evoto.ai/wordpress/8062e5d7-def2-4e89-b43e-3c4a8c19e216.png) **연결된 인물 보정** 그룹 사진 아래에서 가끔은 서로 다른 나이나 성별의 캐릭터를 동시에 조정하길 원할 때가 있습니다. 이 경우에는 이들을 연결하여 한 사진에서 서로 다른 성별이나 연령의 캐릭터를 빠르게 조정할 수 있습니다. Evoto의 새로운 연결된 인물 보정 기능을 사용하면 서로 다른 성별이나 연령의 캐릭터 파라미터를 연결하여 동시에 조정할 수 있는 기능이 가능해집니다. 연결을 적용한 후에는 연결된 캐릭터 유형에 대한 슬라이더를 조정할 수 있습니다. ![](https://res.evoto.ai/wordpress/ffb2d116-0f1c-443c-b802-c044ae55c7c7.png) 연결 아이콘을 클릭하고, 연결하고 싶은 캐릭터를 선택하세요. 모든 연결하고 싶은 캐릭터를 선택하여 자동으로 연결됩니다. **주의: 연결된 모드에서 사진에 프리셋(예: 여성 캐릭터만 편집)을 적용하는 경우, 해당 파라미터는 다른 성별이나 연령의 캐릭터(예: 남성, 어린이 또는 노인 캐릭터)에는 적용되지 않습니다.** **수동 미세 조정** ![](https://res.evoto.ai/wordpress/7e431d70-07a3-47ea-a7cd-2f65ee610585.png) 매끄러움 슬라이더는 각 획의 미세 조정의 양과 투명도의 변화를 제어할 수 있어, 사용자가 스머지 강도를 조절하고 보다 정교한 세부 작업을 수행하며 자연스러운 전환을 완성할 수 있도록 도와줍니다.
