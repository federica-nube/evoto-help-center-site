請依照以下步驟配置您的 OM 系統（Olympus）相機和計算機，以便在 Evoto 中進行線上拍攝。 **檢查電腦連接** Windows: macOS: 確保您的 OM 系統相機設定為正確的連接模式（請參閱以下步驟）。 ![](https://res.evoto.ai/wordpress/b886beec-15d3-4c0b-9d20-5387895acc0f.png) Olympus/OM 系統相機可能無法可靠地連接到 macOS 系統。 為了獲得更穩定的連接，建議： 使用擴展 **USB-C 集線器** 將相機連接到您的電腦。 使用 [Zadig](https://zadig.akeo.ie/) 下載並安裝必要的驅動程式：![](https://res.evoto.ai/wordpress/1d6fa2ab-dcff-4236-bb57-ded31f6d1a8f.png) ![](https://res.evoto.ai/wordpress/735152de-06fe-48e9-8a3c-e1241c68b326.png) 盡可能避免直接連接到 Mac 連接埠。 打開 **Zadig**，轉到 **Options → List All Devices**，然後找到您的相機型號。 ![](https://res.evoto.ai/wordpress/2a591214-7d03-42b7-b3fa-16363f3e651e.png) ![](https://res.evoto.ai/wordpress/3dd7c20b-2351-4166-b3af-ed6201ff2e72.png) 按一下**「取代驅動程式」**開始安裝。 ![](https://res.evoto.ai/wordpress/bf6e0bf9-b7d1-4888-bef2-25959884cf0a.png) ![](https://res.evoto.ai/wordpress/bf7e7a5e-0618-470b-a4be-51e2af98cb1c.png) 等待安裝完成。 安裝后，打開**設備管理員**，並確認您的相機顯示在**攜帶型設備**下。 ![](https://res.evoto.ai/wordpress/d690878c-010f-47d3-b6af-0e9f728ad8f1.png) **配置 OM 系統攝影機設置** 請按照相機上的以下步驟操作，以確保正確連接以進行線上拍攝： **1.USB模式：** 轉到 ****Menu → 設定 → Disp （D4） → USB 模式**** ![](https://res.evoto.ai/wordpress/75b0d5d4-5015-42a1-b336-63acda943493.png) 選擇 **USB/Tether 連接** ![](https://res.evoto.ai/wordpress/6125c1bb-71fe-4ff4-9132-d02962d23cc7.png) **2.背光液晶顯示幕：** 前往**選單 → 設定 → 公用程式 → 背光 LCD** **S**et ****to Hold**** ![](https://res.evoto.ai/wordpress/df23ac74-1c9f-4946-af77-c05609dd47ba.jpg) **3.睡眠：** 前往**選單 → 設定 → 實用程式 → 睡眠** 設置為 ****Off**** ![](https://res.evoto.ai/wordpress/b7b667f5-8f99-4d2b-a2a4-b26a55cb62ce.jpg) **4.自動關機：** 前往**選單 → 設定 → 實用程式 → 自動關機** 設置為 ****Off**** ![](https://res.evoto.ai/wordpress/24b4553e-8e8f-40c5-9c13-ae7cddf89c3a.jpg) **5.快速睡眠模式：** 前往**選單 → 設定 → 實用程式 → 快速睡眠模式** 設置為 ****Off**** ![](https://res.evoto.ai/wordpress/7f238839-acb3-4984-92af-288ab68570b9.jpg)
