# **Atalhos de Teclado**

Novos atalhos de navegação foram adicionados para melhorar a eficiência do seu fluxo de trabalho (com teclado conectado):

- **← / ↑:** Ir para a imagem anterior.
- **→ / ↓:** Ir para a próxima imagem.
- **Segurar as teclas de seta:** Navegação contínua.
- **Command + Tecla de seta:** Ir para a primeira ou última imagem.
- **Ctrl + Tecla de seta:** Pular páginas, movendo ±10 imagens de cada vez.
