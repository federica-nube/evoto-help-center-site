O módulo **Corte** no Evoto para iPad oferece opções de corte flexíveis para atender diversas necessidades de edição, incluindo proporções fixas, dimensões personalizadas e entradas baseadas em resolução.

# **Modos de Corte**

O Evoto oferece três métodos de corte:

- **Proporção Fixa**Selecione entre proporções padrão para manter dimensões proporcionais.
- **Tamanho Livre**Ajuste manualmente a área de corte sem restrições de proporção.
- **L × A × Resolução**Insira valores específicos de largura, altura e resolução para um corte preciso.
  - **Unidade: Polegadas**
  - Ideal para impressão ou requisitos específicos de saída.

Após concluir o corte, toque em **OK** no canto inferior direito para aplicar e salvar o ajuste.

## **Alternância Suave entre Abas**

Durante o corte, o usuário pode alternar entre diferentes abas de edição sem interromper o processo de corte. Isso permite um fluxo de trabalho mais fluido e flexível, facilitando ajustes conforme necessário.

---

# **Opções de Proporção**

- **Original** Mantém as dimensões da imagem conforme foi importada para o Evoto.
- **Conforme Capturado** Recorta com base no tamanho original da captura da imagem.
- **L × A × Resolução** Permite inserir manualmente dimensões exatas e a resolução desejada.

## **Opções adicionais**

- **Botão de Proporção** Toque para alternar rapidamente entre diferentes proporções.
- **Botão de Vincular** Toque para bloquear a proporção enquanto redimensiona a área de corte.

---

# **Tamanhos de Corte Padrão**

O Evoto oferece uma lista de tamanhos padrão comuns para seleção rápida. Estes são ideais para impressão, redes sociais ou outras dimensões específicas de plataformas.
