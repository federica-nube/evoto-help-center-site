Evoto se diseñó para hacer más eficaz la edición de imágenes.
Cuando termine de ajustar una sola imagen, puede sincronizar el efecto con otras imágenes para completar rápidamente el procesamiento por lotes. Método 1 Después de seleccionar una imagen editada, pulse Ctrl + A (Cmd + A para Mac) para seleccionar todas las imágenes y, a continuación, haga clic en Sincronizar con Todas Las Imágenes en la esquina inferior derecha del **Panel de Control** para completar la sincronización por lotes rápidamente. Al sincronizar, puede elegir el efecto que desea sincronizar y, a continuación, hacer clic en "Sincronizar con las imágenes seleccionadas". Método 2 Después de seleccionar una imagen editada, copie los ajustes realizados en ella pulsando Ctrl + C (Cmd + C para Macs), seleccione todas las imágenes y, a continuación, pegue los ajustes en las otras imágenes pulsando Ctrl + V (Cmd + V para Macs). Esto completa el retoque por lotes. Al copiar, puede marcar los efectos correspondientes que deben sincronizarse y hacer clic en "Guardar configuración" para copiar los efectos y sincronizarlos con el resto de las imágenes.
