Evoto proporciona opciones flexibles para personalizar la calidad de tus imágenes exportadas, asegurando que obtengas el resultado deseado cada vez. Esta guía te ayudará a entender cómo optimizar la calidad de imagen y el tamaño de archivo ajustando las configuraciones disponibles.

Para información más detallada, consulta nuestra página completa sobre [Importación y Exportación](https://support.evoto.ai/es/how-to-use-import-export-es/).

---

### Elegir el Formato de Archivo Correcto

Evoto soporta múltiples formatos de imagen, cada uno con un propósito diferente.

- **Formato Original:** Esta opción exporta tu imagen en su formato de archivo original. Si has hecho ediciones a un archivo RAW, se exportará como JPEG. Si la imagen contiene un canal alfa (como un fondo transparente), se exportará como PNG.
- **JPG (8-bit):** El formato ideal cuando el tamaño del archivo es una preocupación, como para uso web, redes sociales y compartir en línea. JPG usa compresión, lo que reduce el tamaño del archivo a costa de algunos detalles de imagen.
- **TIFF (8-bit / 16-bit):** La mejor opción para imágenes de alta calidad sin pérdida. TIFF es un formato profesional que preserva todos los datos de la imagen al evitar la compresión, haciéndolo ideal para edición adicional o impresión profesional y archivado.

---

### Controlar Calidad y Tamaño de Archivo (para Exportaciones JPG)

La configuración de calidad en Evoto está disponible al exportar en formato JPG y te permite equilibrar la calidad de imagen con el tamaño del archivo.

![](https://support.evoto.ai/wp-content/uploads/2025/04/Screenshot-2025-08-15-at-12.07.51-300x285.png)

- **Configuración de Calidad Basada en Porcentaje:** Evoto usa un control deslizante basado en porcentaje para un control preciso sobre la calidad. Puedes elegir entre niveles predefinidos:
- **Bajo, Medio, Alto o Mejor**
- Alternativamente, usa el control deslizante para establecer un porcentaje específico.
- **Nota:** Si estableces el porcentaje demasiado bajo, Evoto aún exportará la imagen con una calidad mínima del 20% para asegurar que la imagen siga siendo utilizable.
- **Limitar Tamaño de Archivo:** Evoto te permite establecer un tamaño de archivo objetivo específico (por ejemplo, "500 KB"). El algoritmo de exportación comprimirá la imagen para acercarse lo más posible a tu tamaño especificado.
- **Importante:** El tamaño final del archivo puede no coincidir exactamente con el tamaño ingresado debido a la naturaleza de la compresión de imagen. El algoritmo hará su mejor esfuerzo para cumplir tus requisitos mientras protege la calidad de imagen.

### Ajustar Resolución y Dimensiones

Cuando recortas o redimensionas una imagen, prestar atención a la configuración de resolución es clave para mantener la claridad.

- **Resolución para Recorte:** Al usar la configuración de recorte **A × A × Resolución**, puedes optimizar la claridad de tu imagen final aumentando el valor de **px/in** (píxeles por pulgada). Por defecto, esto generalmente está configurado en 300, pero puedes aumentarlo hasta 30,000 para asegurar que se mantenga una alta densidad de píxeles durante el proceso de recorte, lo cual es especialmente importante para trabajos de calidad de impresión.

![](https://support.evoto.ai/wp-content/uploads/2025/04/Screenshot-2025-08-15-at-12.08.40-272x300.png)

- **Otras Opciones de Escala:** Para redimensionar, también puedes usar otros tipos de escala como **Porcentaje**, **Ancho y Alto**, o **Borde Largo/Borde Corto** para definir las dimensiones finales de tu imagen exportada.

---

Al ajustar estas configuraciones, puedes lograr el mejor equilibrio entre calidad y tamaño de archivo para cualquier propósito, asegurando que tus imágenes se exporten desde Evoto con la máxima calidad.
