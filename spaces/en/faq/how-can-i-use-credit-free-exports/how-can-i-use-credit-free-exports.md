Users with an**Active Paid Plan**1 now have access to **certain Credit-free Features**2 under the **Color Adjustments**, **Manual Tools**, and **Crop & Rotate** modules with **Eligible Versions**3 that are credit-free to use! Meaning exporting images using only these features will **not** consume credits!

> Note: Users eligible for these benefits are those with an Active Paid Plan 1 , which includes both subscription plans and Pay-As-You-Go credit packages.

# Evoto Version Requirement

**Windows & Mac:** Evoto version **6.1.0 or above** is required.

> Note: The current Credit-free Features 2 are only supported on Evoto Desktop. Versions below 6.1.0 will not be able to access these Credit-free Features 2 . Please update all of your desktop devices to Evoto version 6.1.0 or above on the official website to avoid potential extra charges.

# Export Credit Rules

**For Evoto Desktop 6.1.0 or Above (Active Paid Plan**1**Required):**

- **Free Export:** Exporting images that utilize **only** the Credit-free Features2 listed above will **not** consume credits.
- **Credit Deduction:** Using any feature **outside** of the listed Credit-free Features2 (e.g., AI Color Match, AI Color Looks or any Portrait Retouching tool) will deduct **1 credit per exported image**.
- **Note:** If your initial export uses any credit-required features, the corresponding credit will be consumed and cannot be refunded. Even if you later re-edit and re-export the same image using **only** Credit-free Features2, the credit deducted for the first export remains consumed.

**For All Users on Versions Below 6.1.0 (or Mobile/Instant Versions):**

- **Credit Deduction:** For all users on versions below 6.1.0, on any device (including Evoto Desktop, Evoto iPad, and Evoto Instant), Credit-free Features2 are **not available**. Any editing in the software, regardless of the features used, will deduct **1 credit per exported image**.

**General Re-export Rule:** Regardless of the features used (credit-free or credited), if the original file has not been modified or replaced, re-editing and **re-exporting the same image** within Evoto will **not** deduct additional credits.

---

1 An “Active Paid Plan” includes any valid **Pay-as-you-go** purchase or **subscription plan** that has not expired. 2 “Credit-free Features” are only available to users with an **Active Paid Plan**, allowing them to export images without using credits. 3 “Eligible Versions” refers to the Evoto versions where this feature is available and details can be found in the Evoto Version Requirement.
