#### Exporting Images

When exporting images, you can configure the storage location, image quality format, and color space on the **Export** page.

![](https://evoto.gitbook.io/~gitbook/image?url=https%3A%2F%2F3099291537-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FGcGHhe6Z57mAjGmYtrQ4%252Fuploads%252FJGmvWeab2tHXtCX9PBzo%252FIMG_1033.PNG%3Falt%3Dmedia%26token%3D8ac0a523-3912-4f00-af1d-61419bd40387&width=768&dpr=4&quality=100&sign=bb59edfb&sv=2)

- **Storage Location**Choose to export images to the iPad’s Files app or Photos library.
- **Image Quality Format**Select from:
- **Original Format**
- **JPG**
- **TIFF：**The highest supported export quality is 16.
- **Color Space: sRGB**sRGB is a standardized RGB color space used widely for consistent display across various devices and platforms. It ensures uniform color rendering, especially for screens and online publishing.
- Evoto AI does not alter the original image’s color space.
- If your image is in Adobe RGB, it is recommended to leave this option unchecked to preserve the original color profile.
- If color consistency across devices is a priority, you may check this option to convert the image to sRGB.

#### Cache Settings

To manage cache settings:

**Navigation:** Go to *Personal Center > Settings > Cache Settings.*

- **Auto-Clear Cache**Set the number of days after which cache is automatically cleared.
- **Manual Cache Clearing**Tap Clear Cache to remove stored data manually.

**Note:**Clearing the cache will permanently delete all manual operation records within the app, including:

- Manual liquify adjustments
- Manual brush strokes
- Repair brush actions
