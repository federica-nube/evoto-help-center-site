Hỗ trợ đa ngôn ngữ Evoto trên iPad hiện hỗ trợ nhiều ngôn ngữ khác nhau: Tiếng Việt Tiếng Nhật Tiếng Hàn Tiếng Trung Giản thể Ngôn ngữ mặc định Nếu là lần đầu tải Evoto trên iPad, ngôn ngữ trên ứng dụng sẽ được thiết lập dựa theo ngôn ngữ mặc định của hệ thống. Nếu ngôn ngữ hệ thống không được hỗ trợ trên Evoto iPad (ví dụ: Tiếng Hà Lan), ngôn ngữ trên ứng dụng sẽ được đặt mặc định là Tiếng Anh. Nếu người dùng đã cài đặt ngôn ngữ cụ thể cho Evoto iPad, ngôn ngữ sẽ không thay đổi dù ngôn ngữ hệ thống bị thay đổi. Thay đổi ngôn ngữ trên ứng dụng Để thay đổi ngôn ngữ trên Evoto iPad: Vào Cài đặt trên Evoto → Cài đặt tài khoản → Thay đổi ngôn ngữ ![](https://res.evoto.ai/wordpress/ae13cbc6-f8a5-4b42-af97-9bdbb1cb72eb.png) Bạn sẽ được chuyển hướng đến cài đặt ngôn ngữ của hệ thống. Tại đây, bạn sẽ có thể chọn ngôn ngữ mà bạn mong muốn. ![](https://res.evoto.ai/wordpress/13e1e995-7827-4684-bbeb-ade427aaa86a.png) ![](https://res.evoto.ai/wordpress/d9569ed6-604c-412c-aeba-56e34347544d.png) **Lưu ý:** Để hiển thị tất cả các tùy chọn ngôn ngữ có sẵn trong Evoto khi chuyển đổi ngôn ngữ, ít nhất hai ngôn ngữ phải được thêm vào hệ thống iPad. Nếu chỉ có một ngôn ngữ được chọn trong hệ thống, bạn sẽ cần thêm một ngôn ngữ khác để hiển thị tất cả các tùy chọn ngôn ngữ của Evoto.
