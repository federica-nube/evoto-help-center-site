本指南完整說明如何存取與管理你的 Evoto 帳戶設定，包括個人資訊、安全功能、通知偏好，以及已登入裝置的管理方式。

我要如何進入帳戶設定？#

你可以透過 Evoto 桌面版/iPad App，或 Evoto 官方網站進入帳戶設定。

在 Evoto App 內

  1. 登入 App。
  2. 點擊右上角的頭像（通常是姓名縮寫）。
  3. 從下拉選單中選擇 Account Settings。

Editable Information:

  • Name
  • Phone number
  • Password
  • Linked third-party accounts (if applicable)

在 Evoto 官方網站上

  1. 前往 evoto.ai 並登入。
  2. 點擊右上角的頭像（通常是姓名縮寫）。
  3. 從下拉選單中選擇 My Account。

Editable Information:

  • Name
  • Company name
  • Email notification preferences
  • Language and time zone settings
我要如何更改密碼？#

你可以在 App 或官方網站上更新密碼。

在 Evoto App 內：#
  1. 登入後進入 Account Settings。
  2. 在 Basic Information 區塊點擊 Change Password。
  3. 驗證碼會發送到你註冊的電子郵件或手機。
  4. 輸入驗證碼，接著設定並確認新密碼。
在 Evoto 網站上：#
  1. 進入 My Account，再打開 Security Center。
  2. 輸入目前的密碼。
  3. 使用字母、數字與符號組合建立新密碼。
  4. 點擊 Save 儲存變更。
我要如何更改電子郵件地址？#

你可以透過官方網站更新與 Evoto 帳戶綁定的電子郵件地址。

  1. 登入 Evoto 網站。
  2. 點擊你的頭像並選擇 My Account，再進入 Security Center。
  3. 點擊 Change Email。
  4. 輸入寄送到你目前電子郵件或手機號碼的驗證碼。
  5. 輸入新的電子郵件地址。

Note: The new email address must not already be registered to another Evoto account. If it is, you’ll need to delete that account first.

可用的安全功能有哪些？#

Evoto 提供多種安全功能，幫助保護你的帳戶：

  • Two-Factor Authentication (2FA): If supported, enable 2FA using an authentication app for enhanced protection.
  • Password Strength Indicators: Follow on-screen prompts to create secure passwords.
  • Activity Reviews: Periodically review login history and update your password if you notice any anomalies.
可用的通知與隱私設定有哪些？#

你可以控制接收哪些通知，以及部分與個人資料相關的設定。

Notification Settings:#
  • Customize which alerts you receive, including:
  • Product updates
  • Marketing promotions
  • Account activity
  • Community messages
Privacy Settings:#
  • Control visibility of your profile and activity.
  • Manage data sharing preferences for community and feedback features.

Advanced Tips:

  • Some users fine-tune their notification triggers using community-shared tips. Visit theEvoto Forum for examples.
我要如何管理已登入的裝置？#

你目前的方案會決定可同時登入的裝置數量。

若要管理裝置：

  1. 進入帳戶中的 Device 區塊。
  2. 查看目前已登入裝置的清單。
  3. 在任一裝置旁點擊 Remove，即可將該裝置登出。
  4. 之後如有需要，你仍可重新在該裝置上登入。
當我的圖片上傳到雲端時，資料會存放在哪裡？#

Evoto 的大多數功能都需要將圖片上傳到雲端伺服器進行分析與處理。不過，系統不會永久保存你的資料，所有圖片在處理完成後都會立即刪除。只有以下功能在取得你事先同意的情況下需要儲存：

  1. Cloud Sync & Cloud Storage： 雲端處理：是；雲端儲存：是
  2. AI Color Match： 雲端處理：是；雲端儲存：是
  3. Cropping（ID Photo & AI Horizontal Correction）： 雲端處理：是；雲端儲存：否
  4. Background Replacement： 雲端處理：是；雲端儲存：否
  5. Portrait Retouching： 雲端處理：是；雲端儲存：否
  6. Software Usage（Beta）： 雲端處理：是（選用）；雲端儲存：否

伺服器位置位於美國奧勒岡州。

Evoto 會將我的圖片用於 AI 訓練嗎？#

不會。除非你明確同意，否則 Evoto 絕不會將你的圖片用於 AI 訓練。所有模型訓練資料都來自合法授權的來源：

  1. 授權內容： 從合規的第三方供應商合法購買並取得授權的素材。
  2. 原創攝影： 由 Evoto 聘請的專業攝影師與模特兒拍攝，且 Evoto 擁有完整且獨家的版權。

我們非常重視你的隱私，以及你對自身內容的掌控權。

Evoto 是否符合 GDPR 與其他資料保護法規？#

是。Evoto 致力於在營運的各個層面保護個人資訊。

  1. GDPR / UK GDPR： 已符合
  2. CCPA： 已符合
  3. CH-DSG： 已符合
  4. Japan P-Mark： 已取得認證
  5. ISO 27001： 進行中
  6. SOC II： 審查中
還需要其他協助嗎？#

如果你遇到任何帳戶相關問題，歡迎透過 support@evoto.ai 與我們聯繫，我們很樂意提供協助。
