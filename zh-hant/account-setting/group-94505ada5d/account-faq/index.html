帳戶常見問題

什麼是 Evoto 帳戶？#

Evoto 帳戶能讓你安全地使用我們的照片編輯平台。擁有有效帳戶後，你可以：

  • 在所有支援裝置上登入 Evoto App（電腦版與 iPad）
  • 方便管理點數使用、訂閱方案與帳務設定
  • 使用雲端預設與個人化設定，讓工作流程在不同裝置之間保持一致
  • 接收重要更新、功能公告與官方支援

如何註冊帳戶？#

你可以直接透過 Evoto App 或 Evoto 官網建立新帳戶。

透過 Evoto App 註冊#

  1. 從裝置的 App Store 下載並安裝 Evoto。
  2. 開啟 App，在登入畫面找到 Sign Up。
  3. 輸入你的 Email 或 手機號碼。
  4. 點擊 Sign Up。
  5. 輸入寄送到你 Email 或手機的驗證碼。
  6. 點擊 Log In 完成註冊並進入帳戶。

透過 Evoto 官網註冊#

  1. 前往 evoto.ai。
  2. 點擊右上角的 Sign In。
  3. 你可以輸入 Email 繼續，也可以直接使用 Google / Facebook 快速註冊。
  4. 若使用 Email 註冊，請依照寄到信箱中的確認步驟啟用帳戶。

建議：

  • 請使用你平常會查看的 Email，方便接收帳戶相關通知。
  • 某些地區可能會在註冊時要求額外身份驗證，以提升帳戶安全。
  • 若你想了解更一般性的入門資訊，也可以參考 Start Guide。

疑難排解：沒有收到確認 Email？#

如果你已完成註冊流程，但沒有收到確認信，請依序嘗試以下方式：

1. 檢查垃圾郵件或垃圾信件匣#

確認信有時會被誤判到垃圾郵件匣。請搜尋以下寄件位址：

  • noreply@mail.evoto.ai
  • noreply@mail1.evoto.ai
  • noreply@mail2.evoto.ai
  • noreply@mail3.evoto.ai

如果找到了，請將信件標記為「非垃圾郵件」，避免之後再次被過濾。

2. 將我們加入白名單（安全寄件者清單）#

為了避免後續收信問題，請把上述寄件位址加入聯絡人或白名單。以下是常見信箱服務的做法：

Gmail#
  1. 前往 Settings → See all settings → Filters and Blocked Addresses。
  2. 點擊 Create a new filter。
  3. 在「From」欄位輸入所有 Evoto noreply 位址，可用逗號分隔。
  4. 點擊 Create filter。
  5. 勾選 Never send it to Spam，再完成建立。
Outlook / Hotmail#
  1. 前往 Settings → View all Outlook settings → Mail → Junk email。
  2. 在 Safe senders and domains 下點擊 + Add，逐一加入上述 Email。
  3. 點擊 Save。
Yahoo Mail#
  1. 前往 Settings → More Settings → Filters。
  2. 點擊 Add new filters。
  3. 設定規則，讓來自 Evoto 寄件位址的信件自動進入收件匣。
  4. 點擊 Save。

還是收不到信？#

如果以上方式都無法解決，請聯絡支援團隊：

  • Email： support@evoto.ai

如果我忘記密碼怎麼辦？#

若你忘記 Evoto 帳戶密碼，可以直接在 App 或網站中重設：

  1. 在登入畫面點擊 Forgot Password? 或 Reset Password。
  2. 輸入你註冊的 Email 或手機號碼。
  3. 依照寄到 Email 或手機的指示建立新密碼。

如何登出 Evoto 帳戶？#

若你要登出：

  1. 開啟 Evoto 應用程式。
  2. 點擊右上角或側欄中的帳戶頭像。
  3. 在下拉選單中選擇 Sign Out 或 Log Out。
