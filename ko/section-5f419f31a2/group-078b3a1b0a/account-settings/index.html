계정 설정 및 보안 관리 가이드

wp:tadv/classic-paragraph Evoto 계정의 개인 정보, 보안 기능, 알림 설정, 로그인된 기기 관리를 포함한 다양한 항목을 확인하고 설정하는 방법에 대해 안내합니다.

계정 설정은 어디서 확인하나요?#

Evoto 앱(데스크탑 또는 iPad)과 공식 웹사이트 모두에서 계정 설정에 접근할 수 있습니다.

Evoto 앱 내에서#

  1. 앱에 로그인합니다.
  2. 오른쪽 상단에 있는 아바타(이니셜 또는 프로필 사진)를 클릭합니다.
  3. 드롭다운 메뉴에서 계정 설정을 선택합니다.

수정 가능한 항목:

  • 이름
  • 전화번호
  • 비밀번호
  • 연결된 외부 계정(해당 시)

Evoto 공식 웹사이트에서#

  1. evoto.ai/ko에 접속 후 로그인합니다.
  2. 오른쪽 상단 아바타를 클릭합니다.
  3. 내 아이디을 선택합니다.

수정 가능한 항목:

  • 이름
  • 회사명
  • 이메일 수신 설정
  • 언어 및 시간대

비밀번호는 어떻게 변경하나요?#

앱과 웹사이트 모두에서 비밀번호를 변경할 수 있습니다.

앱에서 비밀번호 변경:#

  1. 로그인 후 계정 설정으로 이동합니다.
  2. 기본 정보 섹션에서 비밀번호 변경을 클릭합니다.
  3. 등록된 이메일 또는 전화번호로 인증 코드가 전송됩니다.
  4. 인증 코드를 입력한 후, 새 비밀번호를 설정하고 확인합니다.

웹사이트에서 비밀번호 변경:#

  1. 내 아이디 → 보안센터로 이동합니다.
  2. 현재 비밀번호를 입력합니다.
  3. 문자, 숫자, 기호를 조합하여 새 비밀번호를 생성합니다.
  4. 변경 사항을 저장합니다.

이메일 주소는 어떻게 변경하나요?#

Evoto 계정에 등록된 이메일은 웹사이트를 통해 변경할 수 있습니다.

  1. Evoto 웹사이트에 로그인합니다.
  2. 아바타 클릭 → 내 아이디 → 보안 센터로 이동합니다.
  3. 이메일 변경을 클릭합니다.
  4. 기존 이메일 또는 전화번호로 전송된 인증 코드를 입력합니다.
  5. 새로운 이메일 주소를 입력합니다.

참고 : 새 이메일 주소는 다른 Evoto 계정에 등록되어 있지 않아야 하며, 이미 등록된 경우 해당 계정을 먼저 삭제해야 합니다.

어떤 보안 기능이 제공되나요?#

Evoto는 다음과 같은 보안 기능을 통해 계정을 보호합니다:

  • 2단계 인증(2FA): 인증 앱을 통해 추가 보안을 설정할 수 있습니다.
  • 비밀번호 강도 표시: 안전한 비밀번호 생성을 위한 시각적 안내가 제공됩니다.
  • 로그인 기록 확인: 주기적으로 로그인 내역을 점검하고, 이상이 있는 경우 비밀번호를 변경하세요.

알림 및 개인정보 설정은 어떻게 하나요?#

사용자는 수신하고자 하는 알림과 일부 개인정보 설정을 직접 조정할 수 있습니다.

알림 설정: 다음과 같은 항목에 대한 수신 여부를 설정할 수 있습니다:

  • 제품 업데이트
  • 마케팅 프로모션
  • 계정 활동
  • 커뮤니티 메시지

개인정보 설정:

  • 프로필 및 활동 공개 여부 설정
  • 공식 홈페이지 > 리소스 > 포럼 및 기능에서의 데이터 공유 여부 관리

꿀팁: 커뮤니티 포럼에서는 알림 트리거를 세부적으로 조절하는 사용자 팁도 공유되고 있습니다.

로그인된 기기는 어떻게 관리하나요?#

계정에서 동시에 로그인할 수 있는 기기 수는 현재 활성화된 플랜에 따라 다릅니다.

기기 관리 방법:

  1. 계정 → 기기 관리 섹션으로 이동합니다.
  2. 로그인되어 있는 기기 목록이 표시됩니다.
  3. 로그아웃하고 싶은 기기 옆의 제거 버튼을 클릭하면 해당 기기에서 로그아웃됩니다.
  4. 필요할 경우, 다시 로그인할 수 있습니다.
