자동 불러오기 & 내보내기

Evoto의 자동 불러오기 및 자동 내보내기 기능은 지정된 폴더를 실시간으로 모니터링하여 새로 추가된 이미지를 자동으로 Evoto에 가져와 처리하고 내보냅니다. 사용자는 이미지를 불러오는 과정에서 프리셋을 자동으로 적용하도록 선택할 수 있고, 사진 보정 작업을 더욱 편리하게 진행할 수 있습니다. 다양한 수요에 따라…

Evoto의 자동 불러오기 및 자동 내보내기 기능은 지정된 폴더를 실시간으로 모니터링하여 새로 추가된 이미지를 자동으로 Evoto에 가져와 처리하고 내보냅니다. 사용자는 이미지를 불러오는 과정에서 프리셋을 자동으로 적용하도록 선택할 수 있고, 사진 보정 작업을 더욱 편리하게 진행할 수 있습니다.

다양한 수요에 따라 사용자는 이미지의 자동 불러오기, 프리셋 효과 자동 적용 및 자동 내보내기를 구현할 수 있으며, 이를 통해 워크플로우를 최적화하고 업무 효율성을 높일 수 있습니다.

자동 불러오기 및 내보내기 기능은 테더링 촬영과 동일한 프로젝트 내에서 동시에 사용할 수 있으며, 두 기능은 서로 간섭하지 않습니다. 테더링 촬영의 캡처 폴더, 자동 불러오기 지정한 모니터링 폴더 및 자동 내보내기로 저장한 폴더는 동일한 폴더로 설정할 수 없습니다. 자동 불러오기 및 내보내기 기능은 현재 프로젝트 내에서만 사용할 수 있으며, 프로젝트 관리 인터페이스로 돌아가면 자동 불러오기 및 내보내기 기능이 중지됩니다.

시작하는 방법#

방법 1 새 프로젝트를 생성하여 새 프로젝트 내에서 자동 불러오기 및 내보내기 기능을 사용할 수 있습니다. [새 프로젝트 페이지]의 [자동 불러오기 및 내보내기]로 이동하여 해당 기능 인터페이스에 접근합니다. 방법 2 작업 화면에 오른쪽 모듈 탭에서 [자동 불러오기 및 내보내기 아이콘]을 클릭하여 기존 프로젝트 내에서 자동 불러오기 및 내보내기 기능으로 들어갈 수 있습니다.

자동 불러오기 및 내보내기 기능을 사용하려면 새 프로젝트를 만드는 것을 권장합니다. 기존 프로젝트의 사진이 다시 자동으로 내보내지는 것을 방지하기 위함입니다.

자동 불러오기 설정#

소스 폴더 - 모니터링하려는 최대 10개 폴더를 선택하여 추가할 수 있습니다. 하위 폴더 포함 옵션을 선택한 경우 자동 불러오기 시 하위 폴더의 새로 추가된 이미지도 불러옵니다. 이미지 유형 불러오기 필터 - JPEG, RAW, TIFF, PNG 네 가지 포맷의 이미지를 자동으로 불러올 수 있습니다. 프리셋 - 사용자 저장된 '내 프리셋'과 시스템 추천 프리셋 중에서 선택할 수 있으며, 가져온 후 자동으로 효과를 적용할 수 있습니다. 이미지 미리보기 사이즈 설정 - 미리보기 창에 어떤 식으로 이미지를 미리보는지 선택할 수 있습니다. 선택한 이미지 유지하거나 최근에 불러온 이미지 보기 두 가지 옵션이 있습니다. 기본값은 최근에 불러온 이미지 보기입니다. 자동 불러오기 시작 - 클릭하면 폴더 내 새로 추가된 이미지를 자동으로 불러오며, 소스 폴더에 원래 있었던 이미지는 자동으로 불러오지 않습니다.

소스 폴더의 하위 폴더 수에는 제한이 없지만, 1단계 하위 폴더 내용만 읽을 수 있습니다. 자동 불러오기가가 시작되면 설정 옵션은 수정할 수 없습니다.

자동 내보내기 설정#

내보내기 위치 - 내보낸 경로 하나만 선택할 수 있습니다. 내보내기 파일 저장 경로는 두 가지가 있습니다 기존 소스 경로 유지: 예를 들어, 원본 폴더 구조가 A-B-이미지인 경우, A를 모니터링하면 A-B-이미지의 구조가 내보내기 경로에 그대로 복사됩니다. 모든 이미지를 동일한 폴더로 내보내기: 모든 이미지를 동일한 내보내기 폴더에 내보냅니다. 자동 내보내기 설정 자동 내보내기 시간 간격: 사용자가 일정 시간 간격마다 자동 내보내기를 실행하도록 설정할 수 있으며, 최소 10초에서 최대 1시간까지 가능합니다. 기본값은 10분입니다. 파일명 지정: 기존 내보내기 설정과 동일합니다 파일 설정: 기존 내보내기 설정과 동일합니다 워터마크: 기존 내보내기 설정과 동일합니다

해당 기능을 활성화하면 프로젝트 내 모든 자동 내보내기 되지 않은 이미지가 내보내집니다. 수동으로 내보낸 이미지도 자동 내보내기가 되지 않아서 다시 내보내지만, 크레딧 이중 과금하지 않습니다. 자동 내보내기를 사용하는 프로젝트에서 non-raw 내보내기 선택하여 수동 내보내기를 하는 것은 권장하지 않습니다. 이는 중복 내보내기 및 요금 부과 문제를 일으킬 수 있습니다. 자동 내보내기가 진행되는 동안 보정 관련 기능이 비활성화됩니다. 프리셋을 업데이트하려면 자동 내보내기를 먼저 취소해야 조정할 수 있습니다. 업데이트된 프리셋은 다시 선택한 후 불러오는 이미지에 적용할 수 있습니다. 자동 내보내기는 원본 이미지의 저장 위치를 변경하지 않습니다. 원본 이미지는 여전히 모니터링된 소스 폴더에 저장됩니다.
