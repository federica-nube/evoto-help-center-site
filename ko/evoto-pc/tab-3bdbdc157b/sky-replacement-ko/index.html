하늘 교체 기능을 사용하면 클릭 한 번으로 하늘이 과도하게 노출되었거나 완벽하지 않은 사진에서 하늘을 자연스럽게 교체할 수 있습니다.

참고: 이미지 내 하늘이 감지되어야만 기능이 작동합니다.

하늘 교체#

  • 추천 배경: Evoto에서 제공하는 전문가 큐레이션 하늘 배경 중 선택

  • 내 이미지 배경: ➕ 아이콘을 클릭하여 개인 하늘 이미지를 업로드
  • 지원 형식: JPG, JPEG, PNG, GIF, TIFF, BMP
  • 최대 용량: 10MB

수동 미세 조정#

하늘 교체 효과가 과도하게 적용된 영역을 복구하거나, 하늘 경계선을 세밀하게 다듬기 위해 수동 조정 펜 도구를 사용할 수 있습니다.

  • 브러시: 효과를 수동으로 추가/제거
  • 다듬기 브러시: 객체의 가장자리를 자동으로 인식하여 보다 정교한 마스크 생성 (머리카락, 섬세한 윤곽에 적합)

하늘 조정#

하늘이 적용된 후, 다음 도구로 하늘의 외형을 세밀하게 조정할 수 있습니다:

  • 수직 위치: 하늘 이미지를 위/아래로 이동
  • 수평 위치: 하늘 이미지를 좌/우로 이동
  • 수평 뒤집기: 하늘 이미지를 좌우 반전
  • 경계 흐림: 하늘과 원본 이미지 간의 전환 경계를 부드럽게 처리하여 자연스럽게 연결

하늘 외형 조절#

  • 색온도: 하늘의 따뜻함/차가움을 조절
  • 색조: 하늘의 색조(초록/마젠타)를 조절
  • 채도: 하늘 색상의 생동감을 조절
  • 밝기: 하늘 전체 밝기 조절
  • 하늘 흐림: 하늘에 피사계 심도 느낌의 흐림 효과 적용
  • 불투명도: 하늘의 투명도 조절

하늘 색상 매칭#

교체된 하늘과 전경(사람, 환경)의 색조를 자연스럽게 맞출 수 있습니다.

  • 물체 색상 매칭:
  • 왼쪽으로 이동 → 원본 색상 유지
  • 오른쪽으로 이동 → 하늘 색상에 맞춤
  • 인물 색상 매칭:
  • 왼쪽으로 이동 → 원본 피부톤 유지
  • 오른쪽으로 이동 → 하늘과 조화롭게 보정

수면 반사 매칭#

이미지에 물이 감지될 경우, 반사 조정 옵션이 활성화됩니다.

  • 하늘 반사: 하늘이 물에 얼마나 반사될지 조절
  • 반사 흐림: 물의 반사에 흐림 효과를 더해 자연스러운 느낌 연출
