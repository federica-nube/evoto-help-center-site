컴퓨터 설정 확인#

For Windows System For MacOS Operation System Windows 시스템에서는 카메라가 올바른 연결 모드로 설정되어 있는지 확인해 주세요. MacOS 시스템에서는 올림푸스 카메라가 맥 컴퓨터와 안정적으로 연결되지 않을 수 있습니다. 더 안정적인 테더링을 연결하기 위해 카메라를 확장 USB C 허브에 연결한 후 컴퓨터에 연결하는 것을 권장드립니다. 올림푸스의 드라이버 소프트웨어를 웹사이트에서 다운로드하여 연결해 주세요.: https://zadig.akeo.ie/ 다운로드 후에는 옵션 -> 모든 장치 목록 -> 해당하는 카메라 모델을 찾아주세요. 드라이버를 교체할 때까지 클릭하여 설치를 완료하세요. 장치 관리자에서휴대용 장치 아래에서 카메라가 컴퓨터에 연결되었는지 확인하세요. 올림푸스 카메라 설정 확인 단계1. 설정으로 이동 -> 디스플레이 (D4)-> USB 모드 선택 -> "USB / 테더 연결" 선택 단계2. 설정으로 이동 -> 유틸리티-> 백라이트 LCD로 이동하여 "Hold"로 설정해 주세요. 단계3. 설정으로 이동 -> 유틸리티 ->슬립으로 이동하여 "Off"로 설정해 주세요. 단계4. 설정으로 이동 -> 유틸리티 -> 자동 전원 꺼짐으로 이동하여 "Off"로 설정해 주세요. 단계5. 설정으로 이동 -> 유틸리티 -> 퀵 슬립 모드로 이동하여 "Off"로 설정해 주세요.
