연결전송 촬영

강력한 초상화 리터칭과 전문적인 테더 촬영 경험을 결합하여 사진 촬영의 효율성을 향상시킵니다.

즉시 촬영 및 사진 확인: 라이브 뷰 스트리밍을 사용하여 Evoto 내부에서 즉시 사진을 확인할 수 있습니다. 복잡한 사진 가져오기 과정을 작별하고 셔터를 누른 후 즉시 사진을 받아보세요. 즉시 촬영 및 후속 작업: Evoto 내부에서 테더 촬영을 연결하는 동안 미리 설정한 것을 즉시 적용할 수 있습니다. 사진 촬영을 마치는 동안 후속 작업 효율성을 높이고, 사진 촬영을 마치는 대로 리터칭을 완료할 수 있습니다. 테더 촬영을 시작하는 방법 방법 1: 프로젝트 작업 공간에서 "새 프로젝트 만들기"를 클릭하고, "테더 촬영"을 클릭하여 테더 촬영을 사용하여 프로젝트를 직접 만들 수 있습니다. 방법 2: 사진 편집 작업 공간에서 테더 촬영을 위한 새로운 기능 섹션을 추가했습니다. 테더 촬영 기능 섹션을 클릭하여 테더 촬영으로 들어갑니다. 현재 테더 촬영은 사진 촬영만 지원하며, 카메라 정보는 제공되지만 조정할 수는 없습니다. 카메라를 트리거하고 사진을 촬영하려면 아이콘을 클릭하십시오. 시작 전에 설정 1단계: 촬영 효과 설정 사진에 효과를 적용해야 할 경우 다음을 선택할 수 있습니다. 이전과 동기화: 이전 사진에 적용한 효과를 현재 사진에 동기화합니다. 프리셋: 현재 사진에 적용할 프리셋을 선택합니다. 2단계: 미리보기 설정 테더 촬영 중에는 두 가지 다른 미리보기 모드 중에서 선택할 수 있습니다. 현재 선택 유지 (기본 선택):이 모드를 선택하면 현재 선택한 사진을 유지하고 자동으로 최신 사진으로 전환되지 않습니다. 가장 최근 촬영된 사진 표시:이 모드를 선택하면 미리보기 사진이 항상 가장 최근에 촬영된 사진으로 유지됩니다. 자동으로 최신 촬영된 사진으로 전환됩니다. 3단계: 파일 이름 지정 원본 파일의 이름을 지정하는 세 가지 방법이 있습니다. 원본 파일 이름 촬영 날짜 - 원본 파일 이름 커스텀 파일명 - 시퀀스 클립보드 콘텐츠 - 시퀀스 테더링 촬영 시, 이름 형식을 '클립보드 콘텐츠- 시퀀스'로 설정하면 촬영된 사진이 자동으로 클립보드 기반으로 이름이 지정됩니다.

클립보드 콘텐츠- 시퀀스 사용 주의사항 이 기능은 클립보드 콘텐츠를 읽고 표시하는 것만 지원합니다. 클립보드 내용은 텍스트만 지원하며, 이미지 인식은 지원하지 않습니다.

사진 이름 지정에 대한 주의사항 사진 이름 지정 시 다음 기호를 사용할 수 없습니다. ';' ':' '^' '' '<' '>' '?' '|' 나열된 기호를 사용한 경우 파일 이름을 지정할 수 없습니다. 4단계: 촬영 저장 위치 셔터를 누른 후 원본 사진을 저장할 위치를 선택하기 위해 저장된 주소를 클릭하세요. Evoto에 의해 후속 처리된 사진은 내보내기할 때 여전히 파일 내보내기 위치에서 선택해야 합니다. 가져오는 파일 타입 새로운 가져오는 파일 타입 필터에는 JPEG 및 RAW 옵션이 제공합니다. 기본 값으로 두 파일 형식을 다 선택되어 있습니다. 하나의 타입만 활성화된 경우, 해당 필터를 비활성화할 수 없습니다. 마우스를 올리면 비활성 상태로 표시됩니다. 현재 Evoto는 자동으로 연결하는 방법을 사용하며, 사용자는 테더 촬영 기능 섹션에서 테더 연결을 수동으로 클릭할 필요가 없습니다. 연결 라인이 카메라와 컴퓨터에 연결되면 자동으로 연결되어 테더 촬영을 활성화합니다. Evoto와 카메라를 연결 해제하려면 연결 해제를 클릭할 수 있습니다. 일부 카메라는 사용자가 성공적으로 연결하려면 카메라 측에서 연결 방법을 선택해야 할 수 있습니다. 테더 촬영 설정 Evoto 설정 Evoto 설정 -> 환경 설정 -> 테더 촬영에 대해 Evoto가 카메라에 자동으로 연결할지 여부를 선택할 수 있습니다. 카메라 설정 카메라 설정을 "자동으로 꺼짐"으로 설정하는 것을 권장합니다. 왜냐하면 카메라가 자동으로 꺼진 경우 Evoto도 컴퓨터에서 연결이 끊어지기 때문에 다시 테더를 활성화하려면 USB 케이블을 다시 연결해야 합니다. 각 브랜드의 카메라마다 다른 카메라 설정이 있습니다. 카메라 브랜드에 따라 해당 카메라 설정을 참고하십시오. 비정상적인 상황 자동 연결 실패* 자동 연결을 여러 번 시도했으나 연결 실패가 발생하는 경우, 자동 연결 설정을 비활성화하고 수동으로 변경해주세요. 사용자는 수동 입력을 통해 테더 촬영을 수동으로 연결할 수 있습니다. 동시에 사용자는 데스크톱에서 테더 라인을 물리적으로 분리했다가 다시 연결할 수도 있습니다. Evoto는 물리적으로 재연결한 후 자동 연결을 시도합니다. 연결 실패의 원인은 다음과 같습니다. 현재 카메라 유형이 Evoto의 테더 촬영에서 지원되지 않는 경우: 카메라 체크리스트를 참조해 테더 촬영을 지원하는 카메라 모델을 확인하십시오. 만약 귀하의 카메라가 체크리스트에 나와 있지 않다면 support@evoto.ai로 연락하여 업데이트를 요청하십시오. 이미 다른 소프트웨어로 테더 촬영을 연결한 경우: 충돌하는 소프트웨어를 닫으십시오. 카메라 연결 문제: 다양한 이유로 카메라가 연결되지 않을 수 있습니다. 카메라를 분리하고 배터리를 제거한 후 다시 설치하고, 카메라를 재시작한 뒤 테더 촬영을 위해 다시 연결해 보십시오. 테더 연결 라인이 불안정한 경우: 어댑터 허브를 사용하는 경우에도 테더 연결에 영향을 줄 수 있습니다. 다른 테더 연결 라인으로 교체하거나 어댑터 허브를 교체하십시오. 일부 어댑터 허브는 데이터 전송을 지원하지 않을 수 있으므로 컴퓨터에 직접 연결하거나 어댑터 허브가 OTG 인증을 받았는지 확인하십시오. Photos, Image Capture, Image Preview 등의 소프트웨어가 카메라 장치를 점유한 경우: 이러한 소프트웨어를 종료하면 테더 촬영이 다시 자동으로 연결됩니다. 테더 연결에 관한 기타 질문이 있으시면 support@evoto.ai로 문의하거나 Evoto 소프트웨어 내의 고객 서비스를 통해 도움을 받으십시오.
