Evoto 프리셋 및 동기화 프리셋은 기준 편집되고 있는 이미지의 값을 한 번의 클릭으로 전부 기록하고, 새 이미지에 바로 적용할 수 있는 기능입니다. 대량의 이미지를 빠르고 훌륭하게 처리해야 하는 경우, 프리셋은 제일 이상적인 도구입니다. 이를 통해 시간을 절약하고 이미지 처리 작업을 보다 효율적이고 쉽고 빠르게 수행할 수 있습니다. Evoto는 현재 프리셋은 '추천'과 '내 프리셋' 두 가지 부분을 구성되고 있습니다. '추천'프리셋 라이브러리는 Evoto에 속하며 Evoto가 공식적으로 업데이트합니다. 저희는 공식 추천 프리셋을 자주 업데이트하고 최적화할 것입니다. '내 프리셋'은 사용자 소유한 프리셋 라이브러리입니다. 분인이 원하는 설정을 저장할 수 있는 공간입니다. 다른 사용자의 프리셋을 가져오려고 하면 프리셋 가져있는 사용자한테 프리셋 토큰을 요청하여 자신의 프리셋 라이브러리로 가져올 수 있습니다. 프리셋 사용 방법 프리셋은 화면 왼쪽에 있는 플로팅 위젯에 있습니다. 원하는 프리셋을 직접 클릭하여 편집 중인 이미지에 적용할 수 있습니다. 새 프리셋 만들기 제어판 섹션 하단에서 '현재 프리셋 저장' 버튼을 찾을 수 있습니다. 프리셋 이름을 입력한 후 프리셋 유형과 프리셋 그룹을 선택합니다. 프리셋으로 저장할 효과를 선택한 다음 '저장'을 클릭하여 새 프리셋 생성을 완료할 수 있습니다. 이전에 있는 프리셋의 효과를 편집한 후 저장할 때 이전 프리셋 효과를 덮어쓰고 싶으면, '교체'를 클릭하고 새 효과의 값을 저장이 됩니다. 이 작업은 새로운 기능 업데이트 및 기존 프리셋의 업데이트에 대해 다 효율적입니다. 프리셋 관리 프리셋 그룹이나 프리셋 자체를 직접 끌어다 놓아 프리셋을 정렬하고 순서를 조정할 수 있습니다. 자주 사용하는 프리셋은 앞쪽으로 이동하여 더 쉽게 찾아서 사용할 수 있습니다. 프리셋 그룹을 다른 그룹으로 끌어다 놓아 프리셋 그룹을 수정할 수도 있습니다. 아이콘은 프리셋 기능의 플로팅 위젯에서 찾을 수 있습니다. 클릭하고 '프리셋 가져오기' 와 '새 그룹 만들기' 버튼을 보일 수 있습니다. 프리셋 가져오기 – 다른 사용자가 제공한 프리셋 값을 토콘으로 복사해서 가져올 수 있습니다. "프리셋 가져오기"를 클릭하고 프리셋 그룹을 선택하면 해당 프리셋을 본인 계정에 가져올 수 있습니다. .XMP 및 .Cube 파일 가져오기 많은 Evoto 사용자가 색상 프리셋을 저장하고 공유하기 위해 .xmp 및 .cube 파일을 사용합니다. Evoto의 이러한 파일 형식 호환성은 이러한 색상 조정을 사용자의 작업 흐름에 쉽게 통합할 수 있도록 합니다. 이는 추가 편집 및 조정이 훨씬 부드럽고 색상 창의력의 가능성을 발휘할 수 있도록 합니다. '프리셋 가져오기'에서 새로운 선택인 로컬 파일 (.xmp/.cube) 가져올 수 있도록 가능합니다. 타입 1. .XMP파일 가져오기 .XMP 파일 가져오면 자동으로 '내 프리셋'에서 한 폴더를 생성됩니다. 타입 2. LUT 정보를 포함하는 .CUBE 파일 또는 .XMP 파일 가져오기 .XMP/.cube 파일을 가져오면 프로필 정보를 포함하는 프리셋이 자동으로 생성됩니다. 필요한 구성 파일이 부족하여 가져온 프리셋이 비정상적으로 작동하거나 올바르게 표시되지 않는 문제를 방지하기 위해, XMP 파일을 가져온 후 Evoto는 해당 프로파일을 지원합니다. 이를 통해 관련 프리셋 파일을 자동으로 인식하고 연결하여 프리셋의 완전한 효과를 보여줍니다. 가져온 XMP 파일이 현재 Evoto에 없는 프로파일을 사용할 경우, 시스템은 누락된 항목을 자동으로 식별하고 사용자가 해당 프로파일을 찾아서 가져오도록 안내합니다. 새 그룹 만들기– 프리셋을 관리하기 위해 새 그룹을 생성합니다. 프리셋 편집은 해당 프리셋에서 마우스 오른쪽 버튼을 클릭하여 편집 가능합니다. 현재 편집을 사용하여 업데이트– 현재 사진의 특성 설정으로 프리셋을 덮어쓰기. 프리셋 공유하기 – 새로운 프리셋 토큰을 생성하여 다른 사람과 공유할 수 있습니다. 다른 계정에서는 본인이 만든 동일한 프리셋 효과를 가져올 수 있습니다. 리네임 – 프리셋 이름 디시 지어주기. 삭제 – 프리셋 삭제하기. Evoto 프리셋에는 4가지 타입이 있습니다 인물 – 인물 보정 기능 모듈의 모든 기능이 있는 경우에만 사용할 수 있습니다. 색상 – 프리셋이 색상 조정 기능 모듈을 적용하는 경우에만 가능합니다. AI 색상– AI 색상 조정(색상 조정을 포함할 수 있음)을 적용하면 변경 사항이 적용됩니다. 조합– 모든 기능 모듈과 조합합니다. 프리셋 공유 기능 최적화 사용자가 프리셋 공유한 코드를 생성한 후, 프리셋이 삭제되거나 업데이트되어도 코드는 계속 사용 가능합니다. 해당 코드에 포함된 프리셋을 계속 가리키고 저장됩니다.
