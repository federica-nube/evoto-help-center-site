이전 이보트 작업 공간은 [라이브러리]와 [보정] 두 섹션으로 업그레이드되었습니다.

[라이브러리]와 [보정] 간에 전환하려면 필요한 탭을 클릭하세요. 라이브러리 인터페이스 소개 내비게이션 섹션 (상단) 내보내기, 내보내기 리스트, 프로필 및 지금 구매 버튼을 찾을 수 있습니다. 라이브러리 페이지에 있을 때는 새 프로젝트 생성 및 새 파일 가져오기 버튼을 확인할 수 있습니다. 프로젝트 관리 섹션 (왼쪽) 프로젝트, 로컬 폴더, 휴지통을 포함합니다. 프로젝트: 프로젝트는 새로 만들기, 탐색 또는 정렬(마지막 접수 시간/프로젝트 이름, 오름차순/내림차순) 기능을 통해 관리할 수 있습니다. 로컬 폴더: 선택한 프로젝트에 포함된 이미지의 저장 경로를 표시하며, 새 폴더를 가져오거나 새로운 하위 폴더를 생성할 수 있습니다. 휴지통: 이미지를 삭제할 때 '휴지통으로 이동'을 선택하면 이곳에 나타납니다. 즉시 삭제하거나 복원할 수 있습니다. 프로젝트 검색 라이브러리 보기의 프로젝트 모듈에서 검색창을 통해 프로젝트 검색을 지원합니다. 검색 시작 방법 다음 방법으로 검색 기능을 실행할 수 있습니다 키보드 단축키： macOS: ⌘F Windows/Linux: Ctrl+F 관리 섹션 (왼쪽 패널)

이 섹션은 프로젝트 구성 및 파일 관리의 중심이 되는 기능들을 제공합니다. 포함 항목: 클라우드 공간: 클라우드 기반 저장 및 동기화를 통해 장치 간 협업과 접근을 원활하게 할 수 있습니다. 자세한 내용은 [클라우드 공간 소개]에서 확인하세요. 프로젝트: 프로젝트를 생성하고, 이름 또는 최근 열람 날짜 기준으로 정렬하여 탐색할 수 있습니다. 관련 폴더: 프로젝트 이미지가 저장된 폴더를 확인하고 관리할 수 있습니다. 폴더 불러오기, 하위 폴더 생성, 이동, 이름 변경, 삭제 등의 작업이 가능합니다. 컬렉션: 여러 프로젝트에 있는 사진들을 원본 파일을 복제하지 않고 그룹화하고 정리할 수 있는 기능입니다. 휴지통: 삭제된 이미지가 보관됩니다. 여기서 복원하거나 영구 삭제할 수 있습니다. 왼쪽 패널의 크기는 사용자의 작업 스타일에 맞게 조정할 수 있습니다. 갤러리 섹션 (하단) 주로 필터링, 보기 전환, 태깅 등 기능을 사용합니다. 필터링: 별 등급, 색상, 깃발, 편집 여부, 내보내기 여부, 형식 및 화면 비율을 사용하여 미리보기 이미지의 레이블을 추가하고 필터링할 수 있습니다. 또한 필름 영역의 오른쪽에서 빠르게 별 등급, 색상, 깃발을 선택하여 레이블 추가할 수 있습니다. 새로운 필터 3종류가 추가되었습니다：파일 이름, 카메라, 카메라 렌즈 정렬: 촬영 시간, 편집 시간, 별 등급, 파일 타입, 파일 크기, 파일명에 따라 오름차순/내림차순으로 정렬할 수 있습니다. 보기: 이미지 표시 확대/축소 비율을 조정하거나 탐색 (비교)보기/크게 보기/ 그리드 보기를 선택할 수 있습니다. 탐색 보기 모드

여러 장의 사진을 선택하여 타일 형태로 나란히 표시하고 직접 비교할 수 있도록 지원합니다. 탐색 보기 진입 방법 단축키 N을 누르거나 ‘탐색 보기’ 버튼을 클릭하면 이 모드로 전환됩니다. 캔버스에는 현재 선택된 이미지가 표시됩니다. 표시 방식: 한 화면에 표시 가능한 최대 이미지는 12장입니다. 여러 이미지를 선택한 후 특정 이미지를 삭제하면, 이후 이미지가 자동으로 빈자리를 채웁니다. 이미지 추가: Cmd+클릭 (Windows: Ctrl+클릭)으로 이미지를 추가할 수 있습니다. 이미지 제거: 이미지의 X 버튼을 클릭하거나 / 단축키로 선택을 해제할 수 있습니다. Evoto에서는 항상 하나의 이미지는 반드시 선택되어 있어야 하기 때문에, 다른 이미지를 모두 제거하면 남은 한 이미지는 해제할 수 없습니다. 마크 지정: 사용자는 클릭 또는 단축키를 통해 별점, 색상 라벨, 선택/거절 표시를 추가할 수 있습니다. 팁 ：다른 모드 전환: G: 그리드 모드로 전환, D: 보정 모듈로 진입, N: 관찰 보기로 진입, L: 큰 이미지 보기 이미지를 더블 클릭하면 큰 이미지 보기로 전환됩니다. 기타: 관찰 보기 모드에서는 이미지 확대/축소가 지원되지 않으며, 해당 단축키도 비활성화됩니다. AI 선별 AI 선별 기능은 라이브러리 작업공간의 '선별' 탭에서 사용할 수 있으며, 수동 및 AI 기반의 선별 기능을 모두 지원합니다. 자세한 내용은 [AI 선별]에서 확인하세요. 빠른 보정 섹션 (오른쪽) - 프리셋

주로 프리셋 섹션을 표시하고 동기화 효과를 제공합니다. 추천: Evoto가 제공하는 프리셋은 추천란에 표시되어, 원하는 보정 효과를 빠르게 이루는 데 도움을 줍니다 팀 : (비즈니스 계정에만 적용) 계정이 특정 Evoto 주 계정과 연동되어 있는 경우, 주 계정에서 제공한 팀 전용 프리셋을 여기서 찾을 수 있습니다. 내 프리셋: 본인이 선호하는 보정 효과를 프리셋으로 저장하여 앞으로 편하게 사용할 수 있습니다. 그룹을 만들어 관리하거나, 다른 사람이 공유한 Evoto 프리셋을 가져올 수도 있습니다. 자신의 프리셋을 다른 사람에게 공유할 수도 있습니다. 동기화: 현재 이미지의 보정 값을 동일 프로젝트 내 다른 이미지에 동기화할 수 있습니다. 현재 지원되는 동기화 값은 AI 색감 보정, 색감 보정, 인물 보정, 배경 보정, 의상 보정, 크롭 & 회전입니다. 메타데이터 섹션

라이브러리 대시보드의 오른쪽에서 메타데이터 모듈을 찾을 수 있습니다. 이를 통해 사용자는 Evoto 내에서 이미지 메타데이터를 직접 보고, 편집하고, 업데이트할 수 있습니다. 기본 정보: 주요 EXIF 및 IPTC 세부 정보를 확인합니다. 일괄 편집: 그리드 보기에서 여러 이미지를 선택하면 메타데이터 변경이 선택된 모든 파일에 적용됩니다. 업데이트를 적용하기 전에 확인 대화 상자가 나타납니다. 메타데이터 업데이트: XMP 사이드카 파일을 통한 가져오기/내보내기를 포함하여 타사 도구와의 호환성을 높이기 위한 전체 메타데이터 읽기/쓰기 작업을 지원합니다. 이미지 미리보기 (중앙) 프로젝트 생성: 현재 Evoto에 프로젝트가 없거나 처음 사용하는 경우, 새 프로젝트를 생성하도록 안내하는 여러 버튼이 표시됩니다. 원하는 옵션을 선택하세요. 미리보기: 새 프로젝트를 생성한 후, 가져온 이미지 등의 미리보기 그림이 화면 중앙에 나타납니다. 이를 자유롭게 드래그하거나 확대/축소할 수 있습니다. 미리보기에는 Evoto 워터마크가 있지만, 내보낸 이미지에는 이 워터마크가 포함되지 않으니 안심하세요. 불러오기 현재 이미지를 가져오는 주요 방법은 세 가지가 있습니다: 가져오기 버튼을 클릭하기: '프로젝트 생성'을 클릭하거나 프로젝트 관리 섹션의 '프로젝트'에서 '➕'를 클릭하여 새 프로젝트를 추가합니다. 상단의 '➕'를 클릭하면 이미지/폴더 가져오기를 지원합니다. '로컬 폴더'에서 '➕'을 클릭한 다음 '프로젝트 생성'를 선택합니다. 드래그 앤 드롭으로 가져오기: 원하는 이미지를 드래그하여 프로그램에 가져옵니다. 오른쪽 클릭하여 가져오기: 마우스 오른쪽 버튼을 사용하여 컴퓨터 로컬에 있는 이미지를 가져올 수 있습니다. 주의사항 비어 있지 않은 프로젝트에 이미지/파일을 가져올 때, 현재 선택한 프로젝트에 가져오거나 새 프로젝트를 생성할 수 있습니다. 비어 있지 않은 프로젝트에 .lrcat 파일을 가져올 경우, 사용자에게 새 프로젝트를 생성할 것인지 또는 가져오기를 취소할 것인지 선택할 수 있습니다.

파일 관리 - 로컬 폴더 로컬 폴더는 프로젝트와 연관되어 있으며, 선택한 프로젝트의 이미지가 있는 폴더만 표시합니다. 로컬 폴더에서는 폴더 가져오기, 하위 폴더 생성, 폴더 이동, 폴더 이름 변경 및 폴더 삭제와 같은 방법으로 폴더를 관리를 지원합니다. 폴더 가져오기: 일괄 가져오기를 지원하지 않습니다. 하위 폴더 생성: 폴더를 선택하지 않으면 기능이 비활성화됩니다. 폴더 삭제: Evoto 내에서만 삭제: 해당 프로젝트에서 해당 폴더와 이미지는 사라지며, 해당 폴더에 있는 이미지가 휴지통에 있을 경우 함께 삭제됩니다. 로컬 디스크에서 삭제: 해당 프로젝트에서 해당 폴더가 사라지며, 다른 프로젝트에 이미지가 있을 경우 '해당 이미지를 찾을 수 없습니다'로 표시됩니다. 폴더 이동: 폴더를 다른 위치로 이동할 수 있습니다. 리네임: 중복된 이름은 사용할 수 없습니다. 주의사항 Evoto는 해당 폴더 내의 이미지를 자동으로 읽지 않으니 안심하세요. 이미지 경로 업데이트 기능을 통해 손실된 폴더 경로를 복구할 수 있습니다. 구버전 Evoto와 새 버전의 로컬 폴더 관리 차이점은 이제 사용자가 'Evoto 내에서만 삭제' 또는 '로컬 디스크에서 삭제' 중 하나를 선택할 수 있다는 점입니다. 이미지 관리 이미지 선택 크게 보기나 그리드 보기 모드 중 어떤 모드에서도 사용자는 마우스를 클릭하여 다른 이미지로 선택 상태를 직접 전환할 수 있습니다. 또한, 단축키를 통해 빠르게 전환하는 것도 지원됩니다. 단축키 - 크게 보기 모드 방향키 왼쪽/위쪽 → 이전 이미지로 이동 방향키 오른쪽/아래쪽 → 다음 이미지로 이동 단축키 - 그리드 보기 모드 방향키 왼쪽 → 왼쪽으로 한 이미지 이동 방향키 오른쪽 → 오른쪽으로 한 이미지 이동 방향키 아래 → 아래로 한 이미지 이동 방향키 위 → 위로 한 이미지 이동 썸네일 확대/축소 조절 다음 방법을 통해 그리드 보기의 줌 레벨을 조절할 수 있습니다: 1. 슬라이더 조절 - 우측 하단의 줌 슬라이더를 드래그 2. 트랙패드 제스처 - 두 손가락을 이용한 핀치 인/아웃 제스처 다중 선택 Shift+클릭 → 첫 번째 이미지를 클릭하고 Shift를 누른 상태에서 마지막 이미지를 클릭하세요. 첫 번째와 마지막 이미지 사이의 모든 이미지가 선택됩니다. Command+클릭 → Command 키를 누른 상태에서 목록의 여러 개별 이미지를 클릭하면 선택한 각 이미지가 강조 표시됩니다. 다중 선택된 이미지 간 전환 - 크게 보기 (L) 왼쪽 화살표 또는 위쪽 화살표 → 왼쪽으로 이동 오른쪽 화살표 또는 아래쪽 화살표 → 오른쪽으로 이동 다중 선택된 이미지 간 전환 - 그리드 보기 (G) 왼쪽 화살표 → 왼쪽으로 이동 오른쪽 화살표 → 오른쪽으로 이동 위쪽 화살표 → 위로 이동 아래쪽 화살표 → 아래로 이동 여러 이미지를 선택한 후, 좌우 화살표 키를 사용하여 이미지 간을 전환하면, 선택된 이미지 내에서 보기 루프가 작동합니다. 선택 범위를 벗어나 위쪽 화살표나 아래쪽 화살표를 누르면 선택이 취소됩니다. 레이블 추가 색상 레이블 선택한 이미지에 대해 4가지 색상 레이블을 지원합니다: 빨간색, 노란색, 녹색 및 파란색. 깃발 레이블 깃발 레이블에는 플래그 지정됨, 거부됨, 없음의 세 가지 상태가 있습니다. 플래그 지정됨: 작은 깃발 아이콘에 체크 표시가 왼쪽 하단 모서리에 표시됩니다. 거부됨: 작은 깃발 아이콘에 X 표시가 왼쪽 하단 모서리에 표시되며 이미지는 회색으로 표시됩니다. 없음: 오른쪽 클릭 옵션에서 선택할 수 있으며, 레이블을 지정할 때 플래그나 거부되지 않은 상태를 의미합니다. 레이블 추가하는 방법 마우스로 사용 가능한 레이블을 선택하고 클릭하세요. 단축키 사용 별 등급 등급 없음 (0) 1점 (1) 2점 (2) 3점 (3) 4점 (4) 5점 (5) 색상 빨간색 (6) 노란색 (7) 녹색 (8) 파란색 (9) 같은 단축키를 눌러 색상 레이블 제거 가능 깃발 없음 (U) 플래그 지정됨 (P) 거부됨 (X) 혹은 이미지를 오른쪽 클릭하고 드롭다운에서 선택하세요. 라이브러리의 그리드 보기 (G)에서만 일괄 레이블 추가 및 편집이 가능합니다. 이미지 필터링 다중 선택 '라이브러리' 또는 '보정'에서 다중 선택 필터를 지원합니다. 필터 아이콘을 클릭하거나 단축키 ()를 사용하여 필터링 섹션을 열거나 닫습니다. 이미지 방향: 가로 사진 / 세로 사진 / 정방형 사진. 필터와 일치하는 이미지가 없으면 이미지가 표시되지 않으며, 이미지를 다시 선택하려면 필터를 지워주세요. 이미지 이동 다음 액션으로 데스크톱의 이미지 파일 경로를 변경합니다. 한 폴더를 다른 폴더로 이동합니다. 하나 이상의 이미지를 다른 폴더로 이동합니다. 참고 사항: 이동 중 이미지에 효과가 처리 중이거나 이미지가 내보내기 중인 경우, 내보내기가 실패할 수 있습니다. 이미지를 대량으로 이동할 때 작업이 중단되면 성공적으로 이동된 이미지는 대상 위치에 남아 있고, 이동에 실패한 파일은 원래 위치에 그대로 남아 있습니다. 이미지 이동의 성공 여부는 원본 이미지 존재 여부, 저장 위치, 권한 및 파일 상태와 같은 요소에 따라 달라집니다. 특별 사례 FAQ 1. 파일을 이동할 수 없는 이유는 무엇인가요? 아래 경로의 파일 폴더는 이동할 수 없습니다. 윈도우: C:\Users\Desktop C:\Users\Document C:\Users\Download C:\Windows C:\Program Files C:\Users\AppData C:\Users\Favorites C:\Windows\System32 C:\Windows\Temp C:\ProgramData C:\Users\Saved Games C:\Program Files (x86) C:$Recycle.Bin C:\Recovery C:$WINDOWS.~BT or C:$WINDOWS.~WS C:\System Volume Information 맥OS: /Users//Desktop /Users//Documents /Users//Downloads /System /Applications /Library /bin, /sbin, /usr /private /Volumes 2. 있는 대상 위치로 폴더를 이동합니다. 동일한 이름과 다른 이름을 동시에 포함하는 파일을 드래그할 때, 이름 충돌이 없는 파일은 성공적으로 이동할 수 있습니다. 현재 이름 충돌이 있는 파일에 대한 팝업은 제공되지 않습니다. 이미지 복사 특정 프로젝트에 이미지를 드래그할 때, 프로젝트 A의 이미지에 있는 효과가 다른 프로젝트로 함께 복사됩니다. 그러나, 개별 이름으로의 이름 변경은 지원되지 않습니다. 이것은 Evoto 내에서만 아니라 로컬 파일 이름을 변경한다는 점을 유의하세요. 라이브러리에서 폴더로 이동 이미지를 저장한 폴더를 보려면 클릭하며, 표시되는 내용이 그에 따라 업데이트됩니다. 이미지 삭제 Evoto 휴지통으로 이동합니다 디스크에서 삭제합니다 보기 모드 그리드 보기 (G) 그리드 보기는 사진을 그리드 셀의 썸네일로 표시합니다. 이 뷰를 사용하여 프로젝트의 모든 이미지 개요를 확인하거나 특정 폴더를 보고 정렬, 조직 및 관리 작업을 용이하게 합니다. 일괄 작업 그리드 보기에서 이미지를 여러 개 선택하여 별표 평점을 수정하거나 프리셋을 적용할 수 있으며, 변경 사항은 선택한 모든 이미지에 적용됩니다. 크게 보기 (L) 단일 사진을 전체 보기로 표시하여 특정
