Welcome to Evoto Mobile!

Experience the perfect blend of professional photography and AI-powered editing. Evoto Mobile supports camera tethering for seamless shooting anywhere, anytime. With a full suite of professional retouching tools, you can refine skin tones, adjust colors, and perfect your images effortlessly.

Professional photography, reimagined for mobile.

Note: Evoto Mobile supports iOS 15 and above only.

Main Page Overview#

No project created yet

1.Profile Center: The Profile Center provides access to essential account functions: View remaining credits, Access export history, Sign out of your account.

2&8. Import Images: Import photos from Photos or Files to create a new project.

3&7. Tethered Shooting: Enable wireless camera connection to import photos directly during your shoot.

4.Learn:View learning resources.

5.Home: Tap to return to the main screen.

6.Project Page:Tap to switch to the Project Page.

Project Created

  1. Tap any photo to open that specific project.
  2. Tap All to view the complete project page.
Project Page Overview#

No Project Created image

When there are no projects, click Create Project or tap the + icon in the upper-right corner to quickly create a new one.

Project created image

Images View

  1. View Switch: Switch to Project View.
  2. Sorting: Displays in Descending order by default; can be switched to Ascending. Layout Adjustment: The number of photos displayed per row ranges from 3 to 6. You can adjust it using the Zoom Inor Zoom Out options, or pinch to zoom with two fingers.
  3. Project Creation Date: Photos are displayed by date. All photos from projects created on the same day are automatically grouped under that date.

Project View#
  1. View Switch: Switch to Photo View.
  2. Create New Project: Enter the project name in the popup window, then choose Import Photos or Tethered Shooting. Import Photos: Select photos from your phone gallery to import. Tethered Shooting: Import photos directly via tethered shooting to create a new project.
  3. Total Photos in Project
  4. Project Name
  5. Last Edited Time: When the project is not empty, all projects are displayed in descending order based on the last edited time.
  6. Project List: All projects are displayed in a 2-projects-per-row layout.
Tethered Shooting Page#

The Tethered Shooting Home displays: camera connection status, a tethered shooting illustration, wired connection guide, wireless connection guide, and a Check button.

  • Camera and network permissions must be granted before tethered shooting.
  • Only one camera can be connected at a time.
Retouch Page Overview#

  1. Back: Return to the previous page.
  2. More Functions: Provides the following options: Library: View all images in the current project; supports Add Photos, Sync, Export, Delete, etc. Sync: Apply the current photo’s edits to other photos. Import Images: Select additional photos from the gallery to add to the current project. Tethered Shooting: Enter the tethered shooting page. Provide Feedback: Report any issues with the current photo. Delete: Delete the current photo. This action cannot be undone, use with caution.
  3. Export: Enter the export page.
  4. Sync: Apply the current photo’s edits to other photos.
  5. Add Images: Import more photos from the phone gallery into the current project.
  6. Effect Comparison: Appears after any edit; tap to view before-and-after comparison. Long press anywhere on the photo to compare.
  7. Thumbnail Area: Displays all photos in the project as thumbnails; supports horizontal swiping to browse.
  8. Library: Shows all photos in the project; supports Import, Sync, Export, and Delete operations.
  9. Feature Modules: Provides access to features like Preset, Color, and Portrait adjustments.
