Personalized AI Look is Evoto's AI-powered feature that allows users to train a custom color grading style by uploading at least 200 edited photos, creating a unique AI Look that matches their personal style.

Training Requirements#
  • Basic: Upload at least 200 color-edited images.
  • Format:

- - RAW Only: 200 RAW images required. - JPEG/TIFF: 200 non-RAW images required. - Both: 200 of each format, 400 total.

  • Color: Must be unified as either color or black & white.
Creation Flow (3 Steps)#
  • Step 1 - Basic Info: Set Look name + Select image type (Wedding/Portrait/Family, etc.)
  • Step 2 - Settings: Select image format (RAW/JPEG/TIFF/Both) + Color type (Color/Black & White)
  • Step 3 - Source Selection: Import from Lightroom Classic or Evoto Project
Image Import#
  • Lightroom Classic: Supports .lrcat file selection, drag-and-drop, and browser import. Filter by folder, collections, star ratings, flags, or color labels.
  • Evoto Project: Select images from projects or collections. Cloud projects require downloading first.
  • AI Smart Selection: Automatically filters images with good diversity and color consistency to improve training. Available when ≥50 images.
Preview & Adjustment#

Import up to 5 images for Look preview. Adjustment tools include: Basic (AI Correction / Fixed Value modes), Curve , HSL , Color Grading , Color Calibration (AI Correction only). - AI Correction: Applies offset correction based on AI-generated values. - Fixed Value: Directly replaces the AI-generated value.

Look Panel#
  • Status: Unready (below minimum) / Ready to Use / Ready to Use (Training) / Locked (exceeded free quota after downgrade).
  • RAW/Non-RAW Looks can only be applied to matching format images.

Note: Looks can be saved as presets. When sharing presets, Look effects are masked (Look sharing not yet supported).
