AIカラーマッチとは？#

AIカラーマッチは、ある画像の色調やスタイルを別の画像に自動的に合わせることができる機能です。参照画像を指定すると、露出、色温度、トーン、局所的なディテールが一貫するように自動調整されます。 AIのセマンティック認識により、肌や髪、衣服、背景（空や室内照明など）も含めて、自然かつスマートに色を整えます。

この機能は、大量の写真や異なる光環境で撮影された画像に統一感を持たせたいユーザーにとって特に有効です。 AIカラーマッチの使い方

ステップ1：AIカラーマッチパネルを開く#

「色補正」パネル内にある「AIカラーマッチ」を開きます。

  • 「AI色補正」機能だけが有効な場合：カラーマッチ機能は「おすすめ」と「マイAIカラールック」タブの両方に表示されます。

  • すべてのパネルが表示されている場合：以下のように、上記と同様に、該当機能は「カラーマッチ」と「マイAIカラーラック」内に表示されます。

ステップ2：参照画像を設定する#

AIカラーマッチに使用する参照画像の設定方法は2つあります：

  1. ローカル画像をアップロード：「参照のアップロード」をクリックして、JPG・TIFF・PNG画像を選択。
  2. 編集中の画像を使用：「編集中の画像から参照画像を作成」をクリックすると、現在表示中の画像が参照画像になります（複数選択されていても可）。

ステップ3：参照色を適用する#

参照画像を設定すると、選択した画像に色補正が自動的に適用されます。

  • 衣服・肌・背景の色が自動的にマッチされます。
  • 参照画像を切り替えることが可能です。

ステップ4：結果を微調整する#

AIカラーマッチ適用後は、以下の2つの方法で結果を調整可能です：

  • 全体調整
  • 適用量
  • トーン（明るさ・コントラスト）
  • 色（全体的な色調）
  • マスク編集
  • 顔・体の肌
  • 髪
  • 目・唇・歯
  • 衣服
  • 背景など

それぞれのマスクごとに色やトーンの強さを個別に調整できます。「リンク」アイコンを使えば全マスクの調整を連動させることもできます。

ステップ5：一括同期またはプリセット保存#

  • 一括同期
  • 設定をコピー（Ctrl+C / Cmd+C）
  • 対象画像を選択
  • 貼り付け（Ctrl+V / Cmd+V）

※AI色補正が有効になっていないと同期されませんのでご注意ください。

  • プリセットとして保存
  • AIカラーマッチと追加調整後、「プリセットを保存」をクリック。
  • 次回以降、同じスタイルを簡単に再利用できます。

AIカラーマッチ：クイックモード vs コントロールモード#

AIカラーマッチ機能がアップグレードされ、より柔軟かつ詳細な調整が可能になりました。

用途に応じて「クイックモード」と「コントロールモード」から選択できます。

クイックモード#

おすすめの用途：自動で素早く色を合わせたいとき

動作の仕組み：

上記でご紹介した従来のAIカラーマッチと同様に、素早くシンプルに自動補正が行われ、追加の調整は不要です。

コントロールモード#

おすすめの用途：参照画像に基づいて細かく色合わせをしたいとき

動作の仕組み：

  • 適用後、自動的にプロファイルが「基本補正」に切り替わります
  • 各スライダーに色補正が直接反映され、調整内容が明確になります
  • AIによる色補正にはローカルマスクが自動作成され、それぞれのマスクに対応した調整が加えられます

リファレンス管理#

「リファレンス管理」機能では、AIカラーマッチのリファレンス画像を色合いやスタイル、または任意のカスタムグループごとに整理できるため、必要なリファレンスにすばやくアクセスして再利用することが可能になります。

アクセス#

「マイリファレンス」パネルの右側にあるボタンをクリックして、「リファレンス管理」ウィンドウを開きます。

リファレンス管理ウィンドウ#

  • 左パネル：ユーザーが作成したリファレンスグループを表示します。グループの作成、名前の変更、削除、および手動での並べ替えが可能です。
  • 右パネル：選択したグループ内のすべてのリファレンスを表示します。リファレンスの一括アップロード、移動、名前の変更、および削除が可能です。

注意事項#

  • 「未分類」グループは常に最上部に表示されるため、順番を並べ替えることはできません。
  • 新規作成、またはインポートしたリファレンスは、デフォルトで自動的に「未分類」グループに追加されます。

ベストな結果のためのヒント#

  • 照明・背景・被写体が似ている参照画像を使用すると精度が上がります。
  • すでにレタッチ済みの画像を参照として使えば、シリーズ全体にスタイルを一括適用可能です。
  • サードパーティのソフトで色補正した場合、.XMP や .CUBE をEvotoにインポートして適用 → その画像を参照に指定することも可能です。

注意点#

  • AIカラーマッチを適用すると、それ以前のカラー編集はリセットされます（スライダーも初期化）。
  • 元に戻すことで編集前の状態に復元できます。
  • AIカラーマッチ適用後に、さらに個別での色補正も可能です。

ワークフロー例#

1. ウェディング撮影：全体トーンの統一

1枚の理想的な編集画像を選び、それを基準にすべての画像にAIカラーマッチを適用 → 必要に応じて微調整 → プリセットとして保存。

2. テザー撮影：撮影中にリアルタイム適用

撮影時にあらかじめ参照画像を準備 → プリセットを作成 → 撮影中の画像に即時適用 → 編集時間を大幅に短縮可能。
